# Colab-Test
Google Colabからのアップロードテスト

## 手順
1. [GitHub](https://github.com/new)で新規リポジトリを作成する。この際、`Add README`をオンにしておく
<img width="540" height="300" alt="pic1" src="https://github.com/user-attachments/assets/1024a5c6-e416-4965-ae3d-37b825b1f029" /></br>

2. [Google Colab](https://colab.research.google.com)でノートブックを開き、メニューバーから`[ファイル] > [GitHub にコピーを保存]`を選択し、GitHubアカウントとColabを連携する
<img width="430" height="500" alt="pic2" src="https://github.com/user-attachments/assets/0516027e-286d-4d32-b371-e89cf9d9fbd3" /></br>

3. `[ファイル] > [GitHub にコピーを保存]`を開き、1.で作成したリポジトリを選択する。（ファイルパスを変更する必要はない）
   コミットメッセージの入力は任意。`OK`をクリックすると、GitHubにプッシュされる。
<img width="755" height="493" alt="pic3" src="https://github.com/user-attachments/assets/12389b0b-8083-4398-830f-6d7f44817bce" />
